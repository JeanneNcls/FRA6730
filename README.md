# FRA6730
Édition critique numérique : Vingt mille lieues sous les mers, Jules Verne

vml_tei.xml : fichier TEI du balisage effectué à partir du texte brut disponible sur Projet Gutenberg à l'adresse https://www.gutenberg.org/ebooks/54873

carte.xsl : feuille XSL permettant de transformer le fichier TEI dans le but d'obtenir la visualisation de la carte. Ce fichier obtenu correspond à index.html, il a été retravaillé manuellement pour correspondre à la visualisation désirée.

html : dossier comprenant l'ensemble des pages html constituant l'édition critique numérique.
